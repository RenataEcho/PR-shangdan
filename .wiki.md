# Project Summary
This project is a comprehensive media account management system designed to streamline the handling of various media accounts for users. It enhances user experience and operational efficiency by providing advanced features for account management, including data storage, user authentication, and integration with external APIs. The system now includes enhanced functionalities for order management and user interaction.

# Project Module Description
The project consists of several functional modules:
- **Authentication**: Manages user login and registration.
- **Media Accounts**: Handles the creation, listing, and management of media accounts, now with edit functionality.
- **Settings**: Allows users to configure their account preferences.
- **Health Check**: Monitors the health status of the application.
- **AI Hub**: Integrates AI functionalities for enhanced media management.
- **Storage**: Manages file storage and retrieval.
- **Order Management**: New module for handling order details and interactions.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Database**: PostgreSQL
- **Deployment**: AWS Lambda for serverless architecture

# Usage
1. Install dependencies for the backend:
   ```bash
   pip install -r app/backend/requirements.txt
   ```
2. Install dependencies for the frontend:
   ```bash
   npm install
   ```
3. Build the frontend application:
   ```bash
   npm run build
   ```
4. Run the backend application:
   ```bash
   python app/backend/main.py
   ```
