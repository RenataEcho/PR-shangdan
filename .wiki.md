# Project Summary
The project is a mobile-oriented task and order marketplace application designed to enhance user engagement with tasks and orders. It features a mobile-first design that allows users to interact intuitively with listings, manage profiles, and conduct transactions. Key functionalities include a rotating banner, streamlined navigation, enhanced task management, improved wallet functionalities, and a consolidated withdrawal feature, all aimed at elevating user experience.

# Project Module Description
The application consists of several functional modules:
- **Index Page**: Displays a feed of available orders with a rotating banner carousel, search, and filter functionalities.
- **Order Detail Page**: Provides detailed information about a selected order, including requirements and settlement rules, along with a task progress tracker.
- **Task Center**: Users can manage and track their tasks associated with orders, featuring a search box, enhanced task cards with commission and progress display, and an optimized overall design. Recent updates include the removal of filter tabs and a change in header stats to reflect completed tasks.
- **Profile Page**: Displays user information and allows management of personal settings, including wallet functionalities such as balance display, transaction history, and withdrawal options.
- **Wallet Records Page**: A sub-page for managing detailed records of income and withdrawals.
- **Wallet Accounts Page**: A sub-page for managing payment methods, including adding and removing accounts.
- **Withdraw Page**: Features a consolidated interface for password input and confirmation in a bottom sheet overlay, enhancing usability and security.
- **Not Found Page**: A fallback for unrecognized routes.

# File Description Inventory
- **README.md**: Overview and setup instructions for the project.
- **package.json**: Lists project dependencies and scripts for building and running the app.
- **src/pages**: Contains all main pages of the application including Index, OrderDetail, TaskCenter, Profile (now includes wallet), WalletRecords, WalletAccounts, and the updated Withdraw page.
- **src/components**: Contains reusable UI components like BottomNav and OrderCard.
- **src/lib/mockData.ts**: Mock data used for populating the application during development.
- **src/hooks**: Custom React hooks for managing state and effects.
- **App.tsx**: The main entry point for the application, setting up routing and context providers.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: A superset of JavaScript that adds static types.
- **Vite**: A modern frontend build tool.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Lucide**: Icon library for React.
- **Sonner**: For toast notifications.
- **React Router**: For routing in the application.
- **TanStack Query**: For data fetching and state management.

# Usage
To get started with the project:
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run lint
   pnpm run build
   ```
