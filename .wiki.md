# Project Summary
The project is a mobile-oriented task and order marketplace application that allows users to browse, manage, and engage with various tasks and orders. It emphasizes a mobile-first design, providing an intuitive interface for users to interact with listings, manage their profiles, and handle transactions seamlessly. The application aims to enhance user experience by offering features such as a rotating banner, streamlined navigation, and improved task management.

# Project Module Description
The application consists of several functional modules:
- **Index Page**: Displays a feed of available orders with a rotating banner carousel, search, and filter functionalities.
- **Order Detail Page**: Provides detailed information about a selected order, including requirements and settlement rules.
- **Task Center**: Allows users to manage and track their tasks associated with orders.
- **Wallet Page**: Displays user account balance, transaction history, and options for withdrawals.
- **Profile Page**: Shows user information and allows management of personal settings.
- **Not Found Page**: A fallback for unrecognized routes.

# File Description Inventory
- **README.md**: Overview and setup instructions for the project.
- **package.json**: Lists project dependencies and scripts for building and running the app.
- **src/pages**: Contains all the main pages of the application such as Index, OrderDetail, TaskCenter, Wallet, Profile, and NotFound.
- **src/components**: Contains reusable UI components like BottomNav and OrderCard.
- **src/lib/mockData.ts**: Mock data used for populating the application during development.
- **src/hooks**: Custom React hooks for managing state and effects.
- **App.tsx**: The main entry point for the application, setting up routing and context providers.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: A superset of JavaScript that adds static types.
- **Vite**: A modern frontend build tool.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Lucide**: Icon library for React.
- **Sonner**: For toast notifications.
- **React Router**: For routing in the application.
- **TanStack Query**: For data fetching and state management.

# Usage
To get started with the project:
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run lint
   pnpm run build
   ```
